# K-Means Clustering Analysis

## Project Overview
This project demonstrates the implementation of the K-Means clustering algorithm, an unsupervised machine learning technique.
K-Means groups data points into K clusters by minimizing intra-cluster variance.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Algorithm Used
- K-Means Clustering
- Elbow Method for optimal K
